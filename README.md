# Copa Mundial — Portal Geral sobre a Copa do Mundo

## Descrição
Página web responsiva que funciona como um portal geral sobre a Copa do Mundo,
reunindo seleções, estádios, campeões e curiosidades da competição. Projeto
desenvolvido para a disciplina de Programação Web I.

## Tema escolhido
Portal geral sobre a Copa do Mundo.

## Tecnologias utilizadas
- HTML5
- CSS3 (estilos próprios em `css/style.css`)
- Bootstrap 5 (grid, navbar, cards, tabela responsiva, componentes de layout)
- JavaScript (interações: menu ativo, botão "voltar ao topo", ano dinâmico)
- Git
- GitHub
- GitHub Pages

## Estrutura do projeto
```
projeto-copa/
```

## Funcionalidades
- Cabeçalho fixo com menu de navegação (Início, Seleções, Estádios, Campeões, Curiosidades)
- Seção principal (hero) com identidade visual do tema
- 4 seções de conteúdo: Seleções, Estádios, Campeões (tabela) e Curiosidades
- 6 cards de seleções + 3 cards de estádios + 4 cards de curiosidades
- Layout 100% responsivo (desktop, notebook, tablet e smartphone)
- Ilustrações próprias em SVG (sem uso de imagens de terceiros)
